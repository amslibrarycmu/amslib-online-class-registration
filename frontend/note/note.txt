ฉันต้องการสร้างเว็บไซต์สำหรับใช้เป็นโปรแกรมลงทะเบียนการเข้าร่วมโครงการอบรมของห้องสมุดคณะเทคนิคการแพทย์ชื่อ AMS Library Couse Registration (AMS-LCR) ซึ่งพัฒนาโดยใช้ React (Version Vite.js) ทั้งหมด พร้อมใช้ mySQL เป็นฐานข้อมูล

สิ่งที่โปรแกรมนี้ต้องมี คือ
1. สามารถสร้างวัน/เดือน/ปีให้แก่ผู้สร้างแบบฟอร์มได้
2. สร้างเป็นเทมเพลตเพื่อใช้ในภายหลังได้
3. สร้างกำหนดการ (Scaldule) ได้อย่างละเอียดด้วยตัวผู้สร้างฟอร์มเอง
4. เมื่อผู้เข้าร่วมได้ลงทะเบียน โดยกด "บันทึก" เพื่อยืนยันและส่งแบบฟอร์มการเข้าร่วมอบรม จะต้องมีการส่งอีเมลเข้าไปที่ผู้ใช้ตามที่ระบุไว้
5. เมื่อใกล้ถึงวันที่จะต้องเข้าร่วมอบรม ให้ส่งอีเมลแจ้งเตือนแก่ผู้เข้าร่วมอีกครั้ง
6. ผู้ใช้ต้องเข้าสู่ระบบผ่าน @cmu.ac.th เท่านั้น
7. การบันทึกข้อมูลผู้เข้าร่วม ให้บันทึกข้อมูลเป็นไฟล์ .csv ลงใน mySQL หรือหากมีความเหมาะสมก็สามารถนำเสนอได้
